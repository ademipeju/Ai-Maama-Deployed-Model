#  AI-Maama: AI-Powered Maternal Risk Prediction

AI-Maama is a smart web application designed to support **pregnant women** and **community healthcare workers**.  
It predicts the risk of **severe bleeding after childbirth** using a machine-learning model.

The system uses a **React frontend** and a **FastAPI backend** with an **XGBoost model** for predictions.
